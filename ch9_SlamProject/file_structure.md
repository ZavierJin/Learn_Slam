# File Structure

```shell
.
├── bin					# Executable binary
├── cmake-build-debug	# for CLion
├── CMakeLists.txt
├── cmake_modules		# Cmake file of third-party library, such as g2o
├── config				# Configuration
├── include				# include "myslam/xxx.h"
├── lib					# Compiled library file
├── src					# Source file, .cpp
└── test				# Test file, .cpp
```
